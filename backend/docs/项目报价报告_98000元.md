## 一、项目概述

### 核心需求
为客户构建AI驱动的挂饰产品设计平台,实现:
- 基于图生图的智能设计生成
- 自然语言对话式编辑
- 批量变体生成
- 成本智能估算
- 自进化优化系统

### 技术方案
- **图像生成**: Nano Banana API (¥0.2/张)
- **语义理解**: Claude API (¥5/M tokens)
- **架构**: FastAPI + PostgreSQL + Redis
- **部署**: 云服务器 + 对象存储
